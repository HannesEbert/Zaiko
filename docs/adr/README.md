# Architecture Decision Records

Significant, hard-to-reverse decisions are recorded here so the reasoning is
written down instead of living only in someone's head. Each record follows the
lightweight format: **Context → Decision → Alternatives considered →
Consequences**.

A decision is never edited after the fact — if it changes, a new ADR
supersedes the old one and the old one is marked as superseded.
